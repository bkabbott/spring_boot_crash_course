# Spring Boot Crash Course

For work, I manage the [Aqua Certify](https://www.youtube.com/watch?v=nlpwegFVJD0&list=PLQkwcJG4YTCSdnQHAHVmGmO7HA6S4a2W3) platform. The backend is currently written in Node.js. It's a pretty basic Express.js API that uses JavaScript.

I regularly run into problems debugging that simply wouldn't happen if I used another language with type.

The code examples in this repo are from [Phillip Lackner's Spring Boot Crash Course With Kotlin](https://www.youtube.com/watch?v=nlpwegFVJD0&list=PLQkwcJG4YTCSdnQHAHVmGmO7HA6S4a2W3) YouTube video.

I am planning on coding our private auth endpoints in Spring Boot with Kotlin and also Nest with TypeScript. Both of these frameworks are opinionated, so they handle architectural decisions in a way that it is generally accepted. From there we will pick a language that has types.
